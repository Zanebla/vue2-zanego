# vue2-zanego

A modern and scalable e-commerce store built with Vue2

# Note

- 项目已经配置自动开启
- eslint 校验功能已经暂时关闭
- src 文件夹别名配置为 @
-

# 基本操作

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
